# 🌿 Trà Chanh Thị Bún – Frontend

Đây là phần giao diện người dùng của web **Trà Chanh Thị Bún** 🍹  
Xây dựng bằng ReactJS + TailwindCSS, có hiệu ứng mượt với Framer Motion ✨

## 🧰 Công nghệ
- ReactJS
- TailwindCSS
- Framer Motion
- Axios
- React-hot-toast
- React-icons

## 🚀 Chạy local

```bash
npm install
npm run dev
